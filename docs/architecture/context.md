# System Context

## Назначение диаграммы

System Context показывает границы Food Ordering Platform и внешние системы, с которыми она взаимодействует.

На этом уровне важно ответить на вопросы:

- кто использует систему;
- какие внешние системы участвуют в бизнес-процессе;
- где проходит граница ответственности Food Ordering Platform.

## Участники

| Участник | Роль |
|---|---|
| Customer | Просматривает меню, оформляет заказ и оплачивает его. |
| Operator | Контролирует заказы и статусы приготовления через веб-терминал. |

## Внешние системы

| Система | Назначение |
|---|---|
| Payment Gateway | Проведение платежей. |
| Warehouse Accounting System | Получение информации об остатках ингредиентов. |
| Robotic Conveyor | Роботизированное приготовление блюд. |
