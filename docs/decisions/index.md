# Архитектурные решения

Архитектурные решения фиксируются в формате ADR — Architecture Decision Record.

ADR нужны, чтобы объяснить не только итоговую схему, но и причины выбора архитектурного подхода.

## Список решений

| ADR | Решение | Статус |
|---|---|---|
| ADR-0001 | Использовать микросервисную архитектуру | Принято |
| ADR-0002 | Использовать API Gateway | Принято |
| ADR-0003 | Использовать Kafka для асинхронных интеграций | Принято |
