# Контейнерная архитектура

Система разделена на клиентские приложения, API Gateway и набор микросервисов. Каждый микросервис владеет своей областью ответственности и собственным хранилищем данных.

Ключевые контейнеры:

- Order Service — корзина и заказы;
- Menu Service — меню;
- Ingredient Service — ингредиенты и остатки;
- Auth Service — авторизация;
- Payment Service — оплата;
- Robot Service — управление приготовлением.
