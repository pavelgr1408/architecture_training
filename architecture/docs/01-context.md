# Контекст системы

Food Ordering Platform — платформа оформления заказов еды. Система предоставляет клиентам возможность просматривать меню, оформлять заказ, оплачивать его и получать статус приготовления.

Внешние зависимости:

- Payment Gateway — проведение платежей;
- Warehouse Accounting System — получение остатков ингредиентов;
- Robotic Conveyor — физическое приготовление блюда.
